## StandardOpenOption
```
StandardOpenOption.CREATE_NEW - gives error while doing operations to file if already exists.
StandardOpenOption.CREATE - Creates a new file if doesnt already exists.
```
## StandardCharsets 
```
    - StandardCharsets.UTF_8: Eight-bit UCS Transformation Format. This is the most common encoding format used on web.
    - StandardCharsets.ISO_8859_1: ISO Latin Alphabet No. 1, covering Western European languages.
    - StandardCharsets.US_ASCII: Seven-bit ASCII encoding.
    - StandardCharsets.UTF_16: Sixteen-bit UCS Transformation Format, with an optional byte-order mark.
    - StandardCharsets.UTF_16BE: Sixteen-bit UCS Transformation Format, big-endian byte order.
    - StandardCharsets.UTF_16LE: Sixteen-bit UCS Transformation Format, little-endian byte order
```
