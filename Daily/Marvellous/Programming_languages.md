## 1. Machine Language (Lowest Level)
```
👉 What it is:
Written in binary (0s and 1s)
Directly understood by the CPU
✔️ Features:
No translation needed
Fastest execution
❌ Drawbacks:
Extremely hard to write and debug
Machine-dependent
```
## 2. Assembly Language
```
👉 What it is:
Uses mnemonics instead of binary
(e.g., MOV, ADD)
Converted to machine code using an assembler
✔️ Features:
Easier than machine language
Still very fast
❌ Drawbacks:
Hardware-specific
Still complex
```
## 3. High-Level Language
```
👉 What it is:
Human-readable languages
Examples: C, Java, Python
✔️ Features:
Easy to write and understand
Portable (runs on many systems)
❌ Drawbacks:
Needs compiler/interpreter
Slightly slower than low-level
```
