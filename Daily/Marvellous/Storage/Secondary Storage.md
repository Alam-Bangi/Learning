🧩 Types of Secondary Storage

1. Magnetic Storage
📀 Hard Disk Drive (HDD)
🔹 How it works:
Stores data using magnetic fields
Uses spinning disks (platters)
✔️ Advantages:
Large storage (TBs)
Cheap
❌ Disadvantages:
Slower
Mechanical parts (can fail)

2. Solid-State Storage
💾 Solid State Drive (SSD)
🔹 How it works:
Uses flash memory (no moving parts)
✔️ Advantages:
Very fast
Durable
Silent
❌ Disadvantages:
More expensive than HDD

3. Optical Storage
💿 CD / DVD / Blu-ray
🔹 How it works:
Uses laser light to read/write data
✔️ Advantages:
Portable
Good for distribution
❌ Disadvantages:
Low capacity
Becoming outdated

4. Flash Storage (Portable)
🔌 USB Drives / Memory Cards
✔️ Advantages:
Small & portable
Easy to use
❌ Disadvantages:
Limited lifespan (write cycles)
